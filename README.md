# 에이블 8기 4차 미니프로젝트 백엔드 입니다.

# 📌 프로젝트명 (Project Title)
도서관리시스템 개발
(AI를 활용한 도서표지 이미지 생성)
## 📝 프로젝트 소개 (Project Description)
해당 프로젝트는 **도서관리시스템**을 위해 개발된 웹/AI 기반 애플리케이션입니다.  
사용자는 **자신이 만든 책의 표지**를 AI를 통해 편리하게 만들수 있습니다.


## 🎯 프로젝트 목표 (Project Goal)
- - Spring Boot + React 기반의 **풀스택 도서 관리 시스템 개발**
- CRUD 기능(등록/조회/수정/삭제) 구현
- OpenAI API(DALL·E)를 연동하여 **도서 표지 이미지를 자동 생성하는 기능** 포함


## 🛠 기술 스택 (Tech Stack)

### **Backend**
- Spring Boot
- Spring Data JPA / Hibernate
- Spring MVC (REST APi)
- Lombok
- JAVA

### **Frontend**
- React
- JavaScript
- React Router
- Material-UI(MUI)

### **Database**
- H2

### **API**
- RESTful API
- OpenAI API (DALL E)

## ✨ 주요 기능 (Features)
- 🔍 도서 등록 <br>
사용자는 책의 제목, 카테고리, 저자, 내용 등을 입력하면 <br>
→ AI가 책 내용을 기반으로 자동으로 표지를 생성해준다.
- 📈 도서 조회 <br>
제목, 카테고리 등 조건을 입력해 등록된 도서를 검색 및 조회할 수 있다.
- ✏️ 도서 수정 <br>
기존 책의 정보를 변경하거나 표지를 다시 생성할 수 있다.
- 🗑 도서 삭제 <br>
더 이상 필요하지 않은 도서를 삭제할 수 있다.
- 🗂 기타 기능들…
- 📝 회원가입(Sign Up) <br>
사용자는 아이디, 비밀번호 등 정보를 입력해 새 계정을 생성할 수 있다.
- 🔑 로그인(Login) <br>
등록된 계정으로 로그인하여 서비스 기능을 이용할 수 있다.
